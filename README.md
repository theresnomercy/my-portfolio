# Professional Portfolio Platform

A developer-focused portfolio system built with Node.js and EJS, designed to showcase technical maturity through modular architecture and clean routing.

## Features

- Server-side rendering (SSR)
- Reusable partials (Navbar, Footer, Head)
- Dynamic project/course management
- Responsive Dark-Mode UI

## Installation & Running

1. Install dependencies:

```bash
npm install
```

2. Start the server:

```bash
npm run dev
```

Then open `http://localhost:3000`.
